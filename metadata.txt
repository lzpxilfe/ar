about=ArchToolkit은 한국 고고학·문화유산 연구 환경에 최적화된 QGIS 분석 도구 모음입니다.
  <br>
  <i>"지식은 전유물이 아닙니다"</i>
  <br>
  <b>도구</b>
  <ul>
    <li><b>DEM 생성</b>: 등고선·표고점 기반 DEM 생성(TIN/IDW 등), 수치지형도(DXF) 코드 프리셋
    <li><b>등고선 추출</b>: DXF 레이어 필터링 또는 DEM에서 GDAL <code>gdal_contour</code> 기반 생성
    <li><b>지적도 중첩 면적표</b>: 조사지역×필지 중첩 면적/비율 계산 + 중첩(클립) 레이어 생성
    <li><b>지형 분석</b>: 경사/사면방향/TRI/TPI/Roughness/Slope Position 분석 + 분류/스타일
    <li><b>경사도/사면방향 도면화</b>: AOI 기준 경사 래스터 + 사면방향(방위각) 화살표 레이어 생성
    <li><b>지형 단면</b>: 다중 프로파일 저장, 지도-차트 연동(hover), AOI/벡터 오버레이, 경사/누적상승/구간 통계, CSV/이미지 export
    <li><b>가시권/가시선(LOS)</b>: 단일/누적/선형/역방향 가시권, (옵션) AOI 가시면적/가시비율 통계, 가중 누적/표준화(0–100%), 히구치 거리대, LOS 프로파일, 곡률·굴절 옵션
    <li><b>비용표면/최소비용경로</b>: DEM 경사 기반 이동 시간/에너지 모델 + LCP + 회랑(Least-cost corridor) + 마찰 레이어(래스터/벡터) + 등시간선/등에너지선(옵션)
    <li><b>최소비용 네트워크</b>: 유적 간 LCP 기반 MST/k-NN/Hub 네트워크 + (옵션) 중심성 지표
    <li><b>근접/가시성 네트워크</b>: 근접성(PPA) 그래프 + DEM 기반 상호가시성(Visibility) 그래프
    <li><b>도면 시각화</b>: 한국 수치지형도(DXF) 레이어 분류/집계 + 카토그래피 스타일 + QML/프리셋 내보내기 + 코드 매핑(JSON) 커스터마이즈 + DEM 배경(옵션)
    <li><b>지구화학도 수치화</b>: WMS RGB → 범례 기반 value/class 래스터 + (옵션) 구간 폴리곤/중심점
    <li><b>AI 조사요약</b>: 조사지역(AOI) + 반경 내 결과 요약 → 보고서 형태 문장 생성 (무료: 로컬 요약 / 옵션: Gemini API)
  </ul>
  대부분의 도구는 실행 중 <b>실시간 작업 로그</b> 창을 통해 진행 상황을 확인할 수 있습니다.
  <br>
  <b>요구 사항</b>
  <ul>
    <li>QGIS 3.40 LTR 이상
    <li>QGIS Processing + GDAL (기본 포함)
  </ul>
  <b>Python 의존성</b>
  <ul>
    <li><code>numpy</code> (대부분의 QGIS 배포판에 기본 포함, 별도 설치 불필요)
  </ul>
