# ArchToolkit 기여 가이드

"지식은 전유물이 아닙니다"

ArchToolkit 프로젝트에 관심을 가져주셔서 감사합니다! 이 프로젝트는 오픈 소스 정신에 따라 누구나 기여할 수 있습니다. 

## 기여 방법

### 1. 버그 제보
버그를 발견하셨다면 GitHub Issues에 상세한 내용을 남겨주세요.
- 어떤 작업을 하다가 발생했는지
- 오류 메시지는 무엇인지 (QGIS 로그 패널 내용 포함)
- 가능하면 재현 가능한 데이터나 스크린샷

### 2. 기능 제안
새로운 아이디어가 있다면 언제든지 제안해 주세요.
- 한국 고고학 연구에 도움이 되는 기능
- 기존 기능의 개선 사항 등

### 3. 코드 기여
직접 코드를 수정하여 기여하고 싶으시다면 다음 절차를 따라주세요.
1. 이 리포지토리를 포크(Fork)합니다.
2. 새로운 브랜치를 생성합니다. (`feature/new-feature` 또는 `bugfix/fix-issue`)
3. 코드를 수정하고 커밋합니다. (커밋 메시지는 명확하게 작성해 주세요)
4. 풀 리퀘스트(Pull Request)를 보냅니다.

## 코딩 규칙
- Python 3.x 환경 및 QGIS 3.x API를 준수합니다.
- 가능한 한 주석을 상세히 달고, 변수명은 의미 있게 작성해 주세요.
- PyQt5/PyQt6 호환성을 고려하여 작성해 주세요.
